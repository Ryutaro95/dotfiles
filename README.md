# dotfiles

個人用設定ファイル

## セットアップ

```bash
./install.sh
```

## 含まれるもの

- **nvim** - Neovim設定
- **tmux** - ターミナルマルチプレクサ設定
- **fish** - Fishシェル設定
- **zed** - Zedエディタ設定
- **wezterm** - ターミナルエミュレータ設定
- **ghostty** - ターミナルエミュレータ設定
- **karabiner** - キーボードリマッピング
- **helix** - テキストエディタ設定
- **starship** - シェルプロンプト

## コマンド

```bash
./install.sh          # dotfilesをインストール
./install.sh --list   # ファイルをプレビュー
./install.sh --uninstall  # シンボリックリンクを削除
```
